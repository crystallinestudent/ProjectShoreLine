I. Installation

How do you have your Ghost server set up? If you don't have one set up, either contact me via http://crystalline-design.com for your own server or host directly through the Ghost team themselves at https://ghost.org.

Follow the instructions below if you are already set up with a server.

Hosted With Ghost

  • Go to your Ghost blog's dashboard and edit your blog.
  • On the right, click Upload a Theme and select the coastal.zip file found within the master zip file.

Self-Hosted

  • Using your favorite FTP client (I use Transmit for OS X) connect to your server and go to the /content/themes folder and extract the coastal.zip file there.
  • Restart your Ghost service. You or your host manager should know how to achieve this if you went the self-hosted route. If not, service ghost restart works.

  • Log in to your Ghost dashboard, go to Settings, scroll to the bottom and switch your theme to Coastal from the dropdown.

II. Theme Setup

As Ghost is still in its beta stages, themes require you to get a little down and dirty to do more indepth editing.

Download a Text Editor

To edit your theme you're going to need a good text editor. I recommend TextWrangler for OS X and Notepad++ for Windows

Change Your Fonts

  • Open the file partials/init/fonts.hbs
  • Coastal uses the Google Fonts Average and Lato by default. In this file, you will see two lines of code in this format:

<link href='//fonts.googleapis.com/css?family=Lato:400,700' rel='stylesheet' type='text/css'>

  • These lines of code can be found on each individual Google Font's page. To see a list of all Google Fonts click here.

Customize Your Menu

  • Open the file partials/menu.hbs
  • You will see a block of code surrounded by {{!Begin Menu Item}} and {{!End Menu Item}}. I recommend ignoring {{!Begin Home Item}} to keep your Home link on your menu.
  • To change the existing menu item, in your code editor add each respective URL in the quotations of the href tag.
  • For example, if I wanted to change it to my static Contact page with the URL of myblog.com/contact, the code would be changed to <a href="{{@blog.url}}/contact">Contact</a>
  • To create new menu items, copy everything in between the {{!Begin Menu Item}} and {{!End Menu Item}} and paste it beneath the existing one, then change the URL and display name as shown in the previous step.

Add Your Social Networks

  • Open the file partials/social-icons.hbs
  • Find the line of code for each social network you'd like to change, for example: <li><a href="" target="_blank" title="Facebook" class="icon fa-facebook-square"></a></li>
  • In your code editor add each respective URL in the quotations of the href tag.
  • For example, if I wanted to add my Facebook URL, the above code would be changed to <li><a href="http://facebook.com/crystallinedesignNC" target="_blank" title="Facebook" class="icon fa-facebook-square"></a></li>

Alternate Post Intro

As of version 1.0.2, the default Post intro is now very similar to the homepage. The minimal Post Intro is still available as the file partials/intro/post-intro-alt.hbs

Switch to Minimal Intro

  • Open the file post.hbs
  • Find the code line {{>intro/intro-post}}
  • Change this to {{>intro/intro-post-alt}}
  • To switch back, simply change it back to the original code.
